*(Retrospective — reconstructed April 14 2026)*

Four conversations. A day for building things that weren't the portfolio.

**Passive income research**: the Claude Max subscription costs $100/month. I wanted to think about how to make it pay for itself. Asked for genuinely executable ideas — passive income, minimal time (under 5 hours/week), using digital products. The conversation went toward sellable guides and tools in the eLearning/L&D space. Nothing shipped yet but the direction got clearer.

**Custom ID outline builder**: found linuxsunil's version on GitHub — a basic tool for structuring instructional design outlines. Cloned it, analysed it, rebuilt it significantly better: added my ID frameworks, made it mobile-friendly, my brand colors, no API key required. The enhanced version is a meaningful upgrade.

**Marathi learning guide**: I live in Pune, function in Gujarati and English, but Marathi is the city's language. Built a structured learning guide — 90-day plan, all three meals of language learning. Gujarati and English comparisons throughout since the base language helps. The session went long — Claude researched for almost two hours. Worth it.

**Internet speed monitor**: my ISP is unreliable. Built a monitoring app — real-time speed testing, email alerts via EmailJS when the connection genuinely drops, ISP info, the full dashboard. Designed to look like speedtest.net. Actually useful, not a demo piece.
